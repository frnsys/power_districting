## Usage

1. Run `gen_graph.py` to generate the tracts graph data.
    - Whenever you update `data.py#load_tracts`, re-run `gen_graph.py`.
    - This produces `gen/graph.json`
2. Run `gen_districts.py` to generate districting proposals.
